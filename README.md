# -# Параллельное умножение матриц (OpenMP, C++)

## Описание проекта

Данный проект реализует алгоритм умножения квадратных матриц с использованием технологии параллельных вычислений OpenMP на языке C++.

Программа автоматически:
- генерирует случайные матрицы
- выполняет их умножение
- измеряет время выполнения
- сравнивает производительность при разном количестве потоков
- рассчитывает ускорение (Speedup)
- сохраняет результаты в файлы
---

##  Конфигурация эксперимента

### Размеры матриц:
- 200 × 200
- 400 × 400
- 800 × 800
- 1200 × 1200
- 1600 × 1600
- 2000 × 2000

### Количество потоков:
- 1
- 2
- 4
- 8

##  Анализ результатов

- Для малых матриц (200–400) параллелизация не даёт прироста  
  → накладные расходы OpenMP превышают выигрыш

- Для средних размеров (800–1200) появляется небольшой прирост  
  → ускорение до ~10%

- Для больших матриц (2000×2000) наблюдается максимальное ускорение  
  → до ~1.16x

###  Вывод:
Параллельное умножение эффективно только при больших объемах данных.

---

## Причины ограниченного ускорения

- накладные расходы на создание потоков
- ограничение пропускной способности памяти
- cache misses (плохая локальность данных)
- отсутствие оптимизаций (например, блочного умножения)
